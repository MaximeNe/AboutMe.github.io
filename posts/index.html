Size of Connex Components in unoriented 2D graph

The project Considering a set of points p_i, 1<i<n, and a limit distance s, we want to know the number a connex component. Two vertices p_i and p_j are linked by an edge if d(p_i, p_j) <= s. A connex component is a sub-graph where all the vertices are linked by transitivity. A solution I worked in pair with ELHAJ-LAHSEN Hugo on different methods.
