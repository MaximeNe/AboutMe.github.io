TodoList implementation

October 1, 2020 · Maxime NEMO

The project#

I always wanted to have a todolist on my desktop (I use Gnome). I made few research for widget on the internet, did not find what I was looking for, so I decided to make my own.
I wanted something that does not consume too much power, that is easy to use (as fast as possible to add something in a terminal) and free.

What I built#

I have chosen to create a todolist that actually changes the background of my computer, so that the consumption of my todolist is down to 0 when I am not changing anything.
The project is available in my github : https://github.com/MaximeNe/TodoList
feel free to check it out
